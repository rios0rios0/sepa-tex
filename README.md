Modelo SEPA da UNIFACS em LaTeX, baseado no **abntex2** para configuracoes basicas.

## Detalhes do Projeto

- **abntex2** utilizada como base para configuracoes basicas
  - [abntex2](http://www.miktex.org/) - pagina do projeto
  - [abntex/abntex2](http://www.tug.org/texlive/) - repositorio do projeto

## Installation

1. Install a LaTeX distribution ([MiKTeX](http://www.miktex.org/) or [TeX Live](http://www.tug.org/texlive/)).
2. Clone the repository:
   ```bash
   git clone https://github.com/rios0rios0/sepa-tex.git
   ```
3. Open the main `.tex` file in your preferred LaTeX editor and compile.
